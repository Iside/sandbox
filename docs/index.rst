Welcome to the dotCloud Sandbox
===============================

dotCloud Sandbox is an open-source project which tries to replicate the sandbox
flavor of dotCloud: a free and easy way to build web applications.

This documentation is divided in two parts, you should definitely read the
user's guide and, if you want to tune something under the hood you should take a
look to the developer guide which covers many internals:

.. note::

   Keep in mind that, right now, this tool is more here to show Docker's
   potential rather than anything else. It's only a few weeks of work and is far
   from covering everything that the dotCloud platform does.
